## 📌 프로젝트 개요

- **🎯 목적:**  
  Nomad Coders 강의를 따라가며 **React 기반의 웹페이지** 개발을 실습

- **🛠 기술 스택:**  
  `React.js`, `CSS`, `JSX`, `React Router`

---

## ✨ 주요 기능

- 🎞️ 영화 목록 카드 UI 렌더링
- 🔍 영화 상세 페이지 (React Router 활용)
- 🎨 CSS로 기본 스타일링 적용
